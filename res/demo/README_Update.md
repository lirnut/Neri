# 更新日志：

## 1.0.1-beta：08月28日，2025年
🎉Neri诞生！项目正式从animal_detect迁移！
<br />
<br />
Neri更新：更新了更接近Neri的主题色；增加了"其他"选项中可增加备注并且随同表格一并导出；表格增加了“物种类别”列，鸟类物种名称基于《中国观鸟年报-中国鸟类名录_v12.0》。
<br />
<br />
模型更新：模型11s_0827.pt:基于YOLO11s，新增鸟类识别（通用模型），将识别到的鸟类归入到"未知鸟"。
